WORK

Mobile App: Daily Dose

A nutrition tracking native mobile app for those with busy lifestyles, Daily Dose uses AI and machine learning to detect the ingredients within a food image and then analyze its nutritional content. It provides a historical log of diet enabling users to make healthier choices.

Built with React-Native with Expo, Redux, Express, PostgreSQL, Sequelize, and utilizing the third-party APIs Clarifai and Edamam.

Github

eCommerce Website

Slime Generation is an eCommerce website dedicated to selling slime products. Visitors to the site can browse through a variety of slime products, add to and update their cart, and checkout their final list of items. The site allows visitors to create an account so that logged-in users can access their order history for previous purchases.

Built in Express, PostgreSQL/Sequelize, React, and Redux. Deployed on Heroku.

Github

Responsive Web Application

Money Tracker is a responsive web application which can track spending and manage budgets built with the NERD stack and using RESTful API architecture. It allows a logged in user to create a budget, create custom categories for the budget, log and edit all spending, and keep track of spending against a specified budget.

Built in Express, PostgreSQL/Sequelize, React, and Redux. Formatted with Bootstrap. Deployed on Heroku.

Github

Heroku

Mobile App: Healthy Choices

Make Healthy Choices is a native mobile application that provides you with quick nutritional takeaways of the food item you are craving, and provides you with a suggestion for a healthier alternative within that food category. Being able to easily compare the nutritional value of both food options will help the user make the healthier choice.

Built in React-Native with Expo, utilizing the third-party API Nutritionix.
